## 2024-04-24 - Input Component Accessibility\n**Learning:** The base Input component was missing automatic ID generation for `htmlFor`-`id` linking, as well as `aria-invalid` and `aria-describedby` for error states, which is a common accessibility trap in custom form components.\n**Action:** Use React's `useId` to auto-generate unique IDs, link labels to inputs, and associate error messages with the input via `aria-describedby`.
