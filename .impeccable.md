# Counsly Design Context

## Users
Tamil Nadu engineering aspirants (ages 17-19) navigating TNEA counselling. They use mid-range Android phones (10k-20k INR, 360px minimum viewport). They are making high-stakes, stressful decisions about their future and need to trust the guidance they receive. The product is priced at 149 INR one-time.

## Brand Personality
**Three words:** Honest, Warm, Unhurried

**Voice:** Like a well-organized guidebook written by someone who has been through this process and genuinely wants to help. No hype, no fake precision, no "AI-powered" claims. Straightforward language that respects the student's intelligence.

**Emotional goal:** Confident clarity. A stressed 17-year-old should feel like "okay, I understand this now" after every screen.

## Aesthetic Direction
**Visual tone:** Warm parchment editorial, inspired by Claude/Anthropic's aesthetic. Paper-like calmness with earthy tones. Serif headings for warmth, sans-serif body for readability, monospace for data.

**Theme:** Light mode only (v2.0). The product is used during daytime, often in shared spaces (home, school, internet cafe).

**References:** Claude/Anthropic's design language (warm grays, serif headings, generous spacing, paper-like surfaces).

**Anti-references:**
- Generic startup look: no gradient heroes, purple-blue palettes, glassmorphism, sparkline decorations
- Cold/corporate feel: no bank-like forms, government portal aesthetics, or clinical sans-serif-only interfaces

## Design Principles

1. **Trust through honesty.** Show broad rank bands, not fake precision. Use "estimated" and "based on historical data." Never overpromise.

2. **Confident clarity.** Every screen should answer "what do I do next?" within 3 seconds. One primary action per screen. No cognitive overload.

3. **Warmth over flash.** Parchment backgrounds, terracotta accents, serif headings. The interface should feel like a well-made paperback, not a tech product.

4. **Mobile-first restraint.** 360px baseline. Dense-but-calm information hierarchy. Touch targets 44x44px minimum. No horizontal scroll. Every primary action reachable within 3 scrolls.

5. **Systematic craft.** Ring-based shadows, tinted neutrals, modular type scale, semantic color tokens. Consistency builds trust.

## Technical Constraints
- Framework: Next.js 16 + React 19 + Tailwind CSS v4 + TypeScript
- No external UI libraries (no shadcn, Radix, Headless UI)
- No dark mode in v2.0
- All fonts: Georgia (serif headings, system), Inter (body, next/font), JetBrains Mono (data, next/font)
- Custom 11-component design system in `src/components/ui/`
